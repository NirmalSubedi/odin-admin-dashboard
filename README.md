# Admin-Dashboard
Replication of an admin dashboard design with mostly CSS grid layout.

**Design To Replicate:**
![Dashboard Design](/imgs/dashboard-project.png)

[**View Replication!**](https://nirmalsubedi.github.io/odin-admin-dashboard/)

## Which skills were demonstrated?
- Laying out main-content, header, and sidebar with css grid.
- Complimenting grid with flexbox to layout the grid-container's items.
- Using the grid property `gap` to create gutters(whitespace) between tracks(columns and rows).
- Spanning sidebar across multiple grid cells with `grid-row` property.
- Utilizing the `repeat()` notation for projects section with `auto-fit` keyword and `minmax()` function to develop responsive grid columns.
- Implementing `fr` - fraction unit to distribute whitespace and create growing and shrinking behavior on grid items.
